# Typography

## Philosophy

Typography should be clean, modern and highly readable.

The hierarchy should immediately guide the user's attention.

Use only two font families throughout the portfolio.

# Heading Hierarchy

Hero Heading
Very large.
Bold.

Section Heading
Large.
Bold.

Project Title
Medium.
Semi-bold.

Card Heading
Medium.

Body Text
Comfortable to read.

Small Text
Muted.

---

# Buttons

Medium weight.

Readable.

Never ALL CAPS unless intentionally used.

---

# Paragraph Width

Avoid extremely wide paragraphs.

Keep line length comfortable.

---

# Spacing

Comfortable spacing between:

Heading

↓

Paragraph

↓

Buttons

Never feel cramped.

---

# Text Alignment

Desktop

Mostly left aligned.

Hero can be center or split depending on design.

Mobile

Readable.

Avoid unnecessary center alignment.

---

# Consistency

Every section should use the same typography scale.

Do not randomly increase or decrease font sizes.

Maintain a consistent visual rhythm.
