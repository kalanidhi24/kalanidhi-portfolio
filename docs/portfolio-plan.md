# Portfolio Plan

## Goal

Create a modern, premium, interactive portfolio that showcases me as a Aspiring Software Engineer whos currently in 3rd year of BE IT Engineer.

The portfolio should immediately communicate professionalism while still feeling unique and memorable.

The design should feel clean, minimal and modern and less of flashy.

The portfolio should prioritize storytelling and engineering projects over 'excessive' animations.
