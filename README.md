# Portfolio Blueprint

This folder contains the complete planning and content for my personal portfolio website.

The goal is to build a modern, responsive, and professional portfolio using React, Tailwind CSS, Framer Motion, and selected ReactBits components.

---

## Folder Structure

docs/
- Design decisions
- Animation rules
- Typography
- Inspiration notes
- Portfolio planning

content/
- Actual content that will appear on the website

assets/
- Profile image
- Resume
- Project screenshots

inspirations/
- Screenshots collected for UI/UX inspiration

website/
- React project source code

---

## Design Principles

- Minimal and modern
- Dark theme
- Smooth animations
- Responsive on desktop, tablet and mobile
- Prioritize usability over excessive effects
- Take inspiration from interactions, not layouts

---

## ReactBits Components

Planned Components

- Dot Grid Background
- Soft Aurora
- Shuffled Text
- Tilted Card
- Chroma Grid

Use these components only where they improve the user experience.

---

## Instructions for AI Coding Assistant

Before generating code:

1. Read everything inside `docs/`.
2. Read everything inside `content/`.
3. Refer to `inspirations/` for UI interactions.
4. Use assets from `assets/`.
5. Build one section at a time.
6. Keep the code modular, reusable, and responsive.
7. Do not copy any inspiration website directly.
8. Preserve the overall design language across all sections.

The goal is to create a unique portfolio that reflects my own style rather than replicating existing portfolios.
