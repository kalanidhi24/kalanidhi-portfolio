## Open Civic Complaint Portal

### Overview

A web-based platform that enables citizens to register, track, and manage civic complaints while helping authorities streamline complaint resolution.

### Tech Stack

- Java
- Spring Boot
- MySQL
- REST APIs

### Key Features

- Complaint registration and tracking
- User authentication
- Complaint status updates
- RESTful API architecture
- Centralized complaint management

### Overview

A MERN-stack application that estimates rainwater harvesting potential, installation costs, and annual savings based on user inputs.

### Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Framer Motion

### Key Features

- Rainwater harvesting estimation
- Cost and savings calculation
- JWT Authentication
- Leaderboard
- Interactive UI
